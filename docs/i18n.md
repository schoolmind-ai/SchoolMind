# Internationalization And Themes

SchoolMind includes server-rendered language and display preferences.

## Supported Languages

- English (`en`)
- Arabic (`ar`, RTL)
- Spanish (`es`)
- French (`fr`)
- Simplified Chinese (`zh`)

The translation dictionary includes full key coverage for supported languages. Spanish, French, and Chinese use simple translations for major UI surfaces and safe English fallback text where a specialized translation has not yet been authored.

## Preferences

Authenticated school users manage preferences at `/settings/account`. Platform owners manage preferences at `/platform/account`. Guests can set language and theme from the public navigation.

Preferences are stored in:

- `user_preferences`
- `platform_admin_preferences`

## Themes And Accessibility

Available themes:

- Light
- Dark
- Midnight
- Calm Blue
- Soft Green
- High Contrast

Display settings include font size, reduced motion, high contrast, and dyslexia-friendly font mode. Settings are applied on the server through `<html>` attributes so the first rendered page uses the saved preference.
